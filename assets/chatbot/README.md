# Chatbot FAB icon

Drop one of these files here to replace the fallback 🤖 emoji on the floating
chat launcher - no other code changes needed, first match wins:

- `robot.webp` (recommended - smallest file size, natively animated, no JS)
- `robot.gif`
- `robot.json` (Lottie - detected, but not yet rendered; see the comment in
  `render_chat_panel()` in app.py for why and what it'd take to add)
